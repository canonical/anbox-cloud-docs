# aar completion

Generate the auto-completion script for AAR for the specified shell.
See each sub-command's help for details on how to use the generated script.

## Usage

    aar completion [command]  

## Flags
  
    -h, --help   help for completion

Use "aar completion [command] --help" for more information about a command.
